# 📱 Mobile Pool — 42 Network

## 🏊 What is a Piscine?

A Piscine is an intensive educational sprint that breaks a subject into daily modules, each introducing new concepts. Every module builds on the last — you learn by doing, failing, and submitting again.

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| React Native | Core mobile framework |
| REST APIs | Live data fetching |
| Firebase / Supabase | Auth & remote database |
| Expo | Development tooling |

---

## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn
- Expo CLI
- iOS Simulator / Android Emulator or a physical device

### Run a Module

```bash
# Clone the repo
git clone git@github.com:kel-baam/mobile.git

# Go into a module
cd mobile/mobileModule00

# Install dependencies
npm install

# Start
npx expo start
```
